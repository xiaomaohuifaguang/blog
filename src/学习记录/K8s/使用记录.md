## 下载安装

### windows

#### kubectl安装
```shell
# 官网教程 https://kubernetes.io/zh-cn/docs/tasks/tools/install-kubectl-windows/
# 创建目录 例如： D:\\k8s
# 下载kubectl.exe 到该目录
# 下载
# 最新稳定版本查询
https://dl.k8s.io/release/stable.txt
# 方式一 如果你已安装了 curl，也可以使用此命令
https://cdn.dl.k8s.io/release/v1.30.0/bin/windows/amd64/kubectl.exe
# 方式二
curl.exe -LO "https://dl.k8s.io/release/v1.30.0/bin/windows/amd64/kubectl.exe"

# 当前目录下执行 验证版本
kubectl version --client
kubectl version --client --output=yaml
# 添加环境变量 Path 追加一个 D:\\k8s
```

#### minikube 安装
```shell
# 官网教程 https://minikube.sigs.k8s.io/docs/start/?arch=%2Fwindows%2Fx86-64%2Fstable%2F.exe+download#Service
# 下载可执行安装包地址https://storage.googleapis.com/minikube/releases/latest/minikube-installer.exe

# 查看所有相关指令
minikube -h
```


### 使用minikube

```shell
# 使用docker作为驱动程序启动
minikube start --driver=docker
# 设置 docker作为默认驱动程序 （可选）
minikube config set driver docker
# 删除
minikube delete
# 面板
minikube dashboard
# 启动一个新的终端，并保持此命令运行
minikube dashboard --url
# 查看节点情况
kubectl get nodes
# 访问集群通过kubectl
kubectl get po -A
# 创建示例部署并将其公开在端口 8080 上
kubectl create deployment hello-minikube --image=kicbase/echo-server:1.0
kubectl expose deployment hello-minikube --type=NodePort --port=8080
# 查看部署应用
kubectl get deployments
# 查看Pod
kubectl get pods
# 查看集群事件
kubectl get events
# 查看 Pod 中容器的应用程序日志
kubectl logs [pod]
# 查看 kubectl 配置
kubectl config view
# 查看信息
kubectl get services hello-minikube
# 访问此服务
# 方式一  让 minikube 为您启动一个 Web 浏览器
minikube service hello-minikube
# 方式二 使用 kubectl 转发端口
kubectl port-forward service/hello-minikube 7080:8080
# 暂停 Kubernetes 而不影响已部署的应用程序
minikube pause
# 取消暂停的实例
minikube unpause
# 停止集群
minikube stop
# 更改默认内存限制（需要重新启动）
minikube config set memory 9001
# 浏览易于安装的 Kubernetes 服务目录
minikube addons list
# 创建运行旧 Kubernetes 版本的第二个集群
minikube start -p aged --kubernetes-version=v1.16.1
# 删除所有 minikube 集群
minikube delete --all
# 导入镜像到minikube中
minikube image load [image.tar]
```
