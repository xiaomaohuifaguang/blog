# 内网穿透

::: tip 摘要
内网穿透，即你的设备没有固定可在公网访问的ip，但是你需要在公网访问
:::

## 使用cpolar

  - 官网地址 [https://www.cpolar.com/](https://www.cpolar.com/)

### linux系统安装  
```shell
# 官网下载 
wget https://static.cpolar.com/downloads/releases/3.3.18/cpolar-stable-linux-amd64.zip
# 解压得到可执行文件 cpolar
unzip /path/to/cpolar.zip
# 连接你的账户 获取token 官网用户设置中 点击 重置您的Authtoken
# 控制台会输出一个配置文件位置 记住以后有用 没记住可以再执行一遍
./cpolar authtoken 你的token 
# 查看帮助
./cpolar help
# 安装服务
./cpolar service install
# 验证
cpolar version
# 开启服务
cpolar service start
# 停止服务
cpolar service stop
```

### 使用方式  
1. 配置文件配置（配置文件为连接账户是创建的 cpolar.yml）
2. UI 页面 服务启动后 默认 Web UI 为 9200 可以访问
